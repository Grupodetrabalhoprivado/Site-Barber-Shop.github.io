# 💈 Barber Shop Premium 💈

Este projeto apresenta um site moderno e responsivo para uma barbearia premium, desenvolvido com foco em uma experiência de usuário elegante e funcionalidades de agendamento avançadas. Inclui um painel administrativo para gerenciamento de conteúdo.

## 🚀 Tecnologias Utilizadas

*   **HTML5**: Estrutura semântica do conteúdo.
*   **CSS3**: Estilização avançada, responsividade (Flexbox, Grid, Media Queries) e efeitos visuais (Glassmorphism).
*   **JavaScript (Vanilla)**: Lógica interativa para agendamento, carrosséis e funcionalidades do painel administrativo.
*   **Google Fonts**: Tipografia elegante (`Playfair Display` e `Inter`).
*   **Font Awesome**: Ícones profissionais para redes sociais e elementos visuais.

## ✨ Funcionalidades Principais

### 🌐 Site Principal (`index.html`)

*   **Design Ultra Premium**: Estética sofisticada com paleta de cores dourada e preta, tipografia aprimorada e fundo temático.
*   **Responsividade Avançada**: Adaptação perfeita a todos os dispositivos (desktop, tablet, mobile).
*   **Sistema de Agendamento Multi-Serviço**: Clientes podem selecionar múltiplos serviços (um por categoria: Corte, Barba, Sobrancelha, Bigode, Pinturas & Tratamentos) em um carrinho temporário.
*   **Carrinho Inteligente**: O carrinho não abre automaticamente; é exibido ao clicar no ícone ou ao finalizar o agendamento.
*   **Seleção de Horários**: Interface intuitiva para escolha de dia e hora, com slots disponíveis e ocupados.
*   **Resumo Detalhado**: Exibe nome do cliente, serviços selecionados, data, horário e valor total antes da confirmação.
*   **Confirmação via WhatsApp**: Gera uma mensagem pré-preenchida com todos os detalhes do agendamento e redireciona para o WhatsApp do barbeiro.
*   **Galeria de Trabalhos**: Carrossel horizontal automático e navegável, exibindo os melhores trabalhos da barbearia.
*   **Seção "O Barbeiro"**: Biografia detalhada com foto (efeito Glassmorphism), experiência e links diretos para WhatsApp e Instagram.
*   **Botões Animados**: Botões de agendamento e confirmação com animações interativas para uma melhor experiência do usuário.

### ⚙️ Painel Administrativo (`admin.html`)

*   **Interface Intuitiva**: Layout moderno com navegação por seções (Dashboard, Serviços, Galeria, Barbeiro, Configurações).
*   **Gerenciamento de Serviços**: Adição, edição e exclusão de serviços com **upload de imagens locais** e pré-visualização.
*   **Gerenciamento de Galeria**: Upload de imagens locais para o portfólio com pré-visualização.
*   **Edição do Perfil do Barbeiro**: Atualização de biografia, foto (com upload local) e estatísticas.
*   **Configurações Gerais Organizadas**: Gerenciamento de informações de contato, redes sociais e horários de funcionamento em abas dedicadas.
*   **Dashboard (Simulado)**: Visão geral de métricas importantes (agendamentos, faturamento), preparado para integração com backend real.

## 🛠️ Próximos Passos (Desenvolvimento Futuro)

Para um funcionamento completo e em produção, as seguintes etapas são recomendadas:

1.  **Implementação de Backend**: Desenvolver uma API RESTful e integrar um banco de dados para persistência de dados (serviços, agendamentos, configurações).
2.  **Autenticação e Autorização**: Adicionar um sistema de login seguro para o Painel Administrativo.
3.  **Hospedagem**: Publicar o site e o backend em um servidor web.
